Welcome

Google Cloud Platform

In this workshop we will be using Google Cloud Platform (GCP). GCP provides both a managed Kubernetes cluster: Google Kubernetes Engine, and a hosted command line environment: Google Cloud Shell.

Cloud Shell is a Debian-based virtual machine is provisionned with all the development tools you’ll need (Kubernetes, Docker, Python, Java, Node, etc.). It offers a persistent 5GB home directory.

This means that all you will need for this workshop is a browser.

Let’s start by connecting to GCP Console. Log in and open your project.

Setup Cloud Shell

  • Open Cloud Shell (on the upper-left corner)

  • Set default project

    gcloud config set project $USER-k8s-istio-workshop

  • Set default region/zone

    gcloud config set compute/region europe-west1
gcloud config set compute/zone europe-west1-b

  • Clone the repository

    git clone https://github.com/Sfeir/kubernetes-istio-workshop.git
cd kubernetes-istio-workshop
